¿Que es Java?

Java es un lenguaje multiplataforma, orientado a objetos y centrado en la red que se puede utilizar como una plataforma en sí mismo.


Además, es un lenguaje de programación rápido, seguro y fiable para codificar todo, desde aplicaciones móviles y software empresarial hasta aplicaciones de macrodatos y tecnologías del lado del servidor.

Caracteristicas de Java

Java es uno de los lenguajes de programación más populares porque ha sido diseñado para facilitar su uso. Algunas razones por las que los desarrolladores profesionales siguen eligiendo Java sobre otros lenguajes de programación incluyen:


  • Recursos de Aprendizaje de Alta Calidad: Java existe desde hace mucho tiempo, por lo que hay muchos recursos de aprendizaje disponibles para los nuevos programadores. La documentación detallada, los libros completos y los cursos ayudan a los desarrolladores a lo largo de la curva de aprendizaje. Además, los principiantes pueden comenzar a escribir código en Core Java antes de pasar a Advanced Java.

  • Funciones y Bibliotecas Incorporadas: Al usar Java, los desarrolladores no necesitan escribir cada nueva función desde cero. En cambio, Java proporciona un rico ecosistema de funciones y bibliotecas integradas para desarrollar una variedad de aplicaciones.

  • Apoyo Comunitario Activo: Java tiene muchos usuarios activos y una comunidad que puede ayudar a los desarrolladores cuando enfrentan desafíos de codificación. El software de la plataforma Java también se mantiene y actualiza periódicamente.

  • Herramientas de Desarrollo de Alta Calidad: Java ofrece varias herramientas para admitir la edición automatizada, la depuración, las pruebas, la implementación y la administración de cambios. Estas herramientas hacen que la programación de Java sea más rápida y rentable.

  • Plataforma Independiente: El código Java puede ejecutarse en cualquier plataforma subyacente, como Windows, Linux, iOS o Android, sin tener que volver a escribir. Esto lo hace especialmente poderoso en el entorno actual, donde queremos ejecutar aplicaciones en múltiples dispositivos.

  • Seguridad: Los usuarios pueden descargar código Java que no sea de confianza a través de una red y ejecutarlo en un entorno seguro en el que no puede causar ningún daño. El código que no es de confianza no puede infectar el sistema host con un virus ni puede leer o escribir archivos del disco duro. Los niveles de seguridad y las restricciones en Java también son altamente configurables.

Principales Usos de Java

Debido a que Java es un lenguaje versátil y de uso gratuito, crea software localizado y distribuido. Algunos de los Principales usos que se le brinda a Java son:


  • Desarrollo de juegos: muchos juegos móviles, de ordenador y de video populares se crearon en Java. Incluso los juegos modernos que integran tecnología avanzada, como el machine learning o la realidad virtual, se crean con tecnología Java.

  • Computación en la nube: Java a menudo se conoce como WORA: escribir una vez y ejecutar en cualquier lugar (por sus siglas en inglés “Write Once and Run Anywhere”), lo que lo hace perfecto para aplicaciones descentralizadas basadas en la nube. Los proveedores de la nube eligen el lenguaje Java para ejecutar programas en una amplia gama de plataformas subyacentes.

  • Macrodatos: Java se usa para motores de procesamiento de datos que pueden trabajar con conjuntos de datos complejos y cantidades masivas de datos en tiempo real.

  • Inteligencia artificial: Java es una fuente inagotable de bibliotecas de machine learning. Su estabilidad y velocidad lo hacen perfecto para el desarrollo de aplicaciones de inteligencia artificial como el procesamiento del lenguaje natural y el aprendizaje profundo.

  • Internet de las cosas: Java se ha utilizado para programar sensores y hardware en dispositivos de borde que pueden conectarse de forma independiente a Internet.

Entrada y Salida de Datos por Consola

En Java al igual que en todos los lenguajes de programación es posible interactuar mediante la consola, a continuación se ejemplificara como generar salida y entrada de datos por consola en Java.


Para enviar un mensaje por consola se usa la siguiente sintaxis:

public static void main(String[] args) {
System.out.println("mensaje por consola");
};

Para recibir entrada de datos por consola se usa la siguiente sintaxis:

/* Primero se importa la libreria java.util.Scanner */
import java.util.Scanner;

public class Entrada {

public static void main(String[] args) {

Scanner input = new Scanner();
System.out.print("Ingrese un número: ");
int numero = input.nextInt();
System.out.println("El número ingresado fue: " + numero);
}
}

Funciones / Metodos

Las funciones o métodos son un bloque de código el cual representa una acción o funcionalidad que poseen un conjunto de objetos. Sin embargo, como Java es un lenguaje de programación fuertemente orientado a objetos, las funciones por si solas no existen, sino que se les denomina métodos debido a que los dos bloques fundamentales de una clase son los atributos y los métodos.


Algunas caracteristicas generales que deben poseer los métodos son:

  1. Las funciones o métodos pueden tener o no tener retorno
  2. Por convencion, el nombre de los métodos deben declararse usando las notación "camelCase"
  3. Por convención o buenas prácticas a la hora de desarrollar software en Java, el primer método que debe declararse es el método constructor
  4. Las funciones o métodos pueden recibir o no argumentos, es decir, pueden declararse con o sin parámetros

A continuación se presentan 3 ejemplos de funciones o métodos en Java:

Funcion Sumar que recibe dos números como argumentos y suma los mismos

public double sumar(double num1, double num2) {
double suma = num1 + num2; return suma;
};

Métodos set y get del atributo String nombre, de una clase denominada Persona

public void setNombre (String nombre) {
this.nombre = nombre;
};
public String getNombre () {
return this.nombre;
};

Funcion que hace que un objeto perteneciente a la clase Persona se presente

public String presentarse() {
String presentacion = "Hola mi nombre es " + this.nombre + " y tengo " + this.edad + ". Es un placer conocerte"
};
